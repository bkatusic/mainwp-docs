This article explains how to get MainWP Essentials and activate your API key on your MainWP Dashboard, which allows you to install add-ons directly and enable automatic updates.

## What You'll Learn

- How to create a MainWP.com account
- How to get MainWP Essentials (free)
- How to copy and activate your API key
- How to install free add-ons

---

## Create Your MainWP.com Account and Get MainWP Essentials

<Steps>
<Step title="Open the free bundle checkout link">
Use the [MainWP Essentials checkout link](https://mainwp.com/checkout/?add-to-cart=1264797&empty-cart). This adds the free bundle to your cart automatically.
</Step>
<Step title="Create your account during checkout">
On the checkout page, enter your new account username and password.
</Step>
<Step title="Place the free order">
Complete checkout to create your MainWP.com account and activate MainWP Essentials.

<Note>There is no charge and no payment method is required for MainWP Essentials.</Note>
</Step>
</Steps>

## Copy Your API Key

<Steps>
<Step title="Navigate to your Account Extensions page">
Navigate to your **[Account > Extensions](https://mainwp.com/my-account/my-api-downloads/)** page. Here you will find your Main API Key.
</Step>
<Step title="Copy your Main API Key">
Copy your Main API Key - you'll need it in the next section.

![MainWP main API key in the My Account page.](/images/miscellaneous/advanced-miscellaneous-4057bc.png)
</Step>
</Steps>

## Activate Your API Key in MainWP Dashboard

<Steps>
<Step title="Navigate to your MainWP Dashboard site">
Go to your self-hosted MainWP Dashboard site.
</Step>
<Step title="Go to MainWP > Add-ons">
Navigate to the **MainWP > Add-ons** page.
</Step>
<Step title="Open the license controls">
Click **Add License Key** (or **Manage License** if a key is already saved).
</Step>
<Step title="Paste your API Key">
Paste your Main API Key and check **Remember Key**.
</Step>
<Step title="Validate your API Key">
Click **Validate License**.

![Click the Validate License button](/images/miscellaneous/advanced-miscellaneous-785926.gif)
</Step>
</Steps>

## Install Add-ons

<Steps>
<Step title="Click Install Add-ons">
Click the **Install Add-ons** button.
</Step>
<Step title="Select and install your add-ons">
Select the add-ons you want to install and click the **Install Selected Add-ons** button.

![Extension selection modal showing available free extensions](/images/miscellaneous/advanced-miscellaneous-a0b5e2.png)
</Step>
</Steps>
