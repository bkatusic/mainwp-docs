The Database Updater Extension detects available database updates for WooCommerce, Elementor, and Elementor Pro. This means that you no longer need to access the WordPress admin area of a child site to start these database updates.

## Install the Database Updater Extension

<Steps>
<Step title="Go to your MainWP Dashboard">
</Step>
<Step title="Go to Add-ons > Manage Add-ons">
</Step>
<Step title="Click the Install Add-ons button">
</Step>
<Step title="Locate the Database Updater extension and select it">
</Step>
<Step title="Click the Install Selected Add-ons button">
</Step>
<Step title="Activate the add-on after installation">
Once the installation process is complete, click the **Activate Add-ons** button.
</Step>
</Steps>


## How to Update the Databases

After you've installed the extension, a Database Updates tab becomes available. Open it from **Sites > Updates > Database**. You can also select **Add-ons > Database Updater**, which routes to the same view.

### Update databases on all child sites

<Steps>
<Step title="Go to Sites > Updates > Database">
</Step>
<Step title="Click Update All for the plugin you want to update">
</Step>
</Steps>


### Update databases on individual child sites

<Steps>
<Step title="Go to Sites > Updates > Database">
</Step>
<Step title="Expand the plugin row and locate the desired site in the table">
</Step>
<Step title="Click the Update Now button next to the desired site">
</Step>
</Steps>
